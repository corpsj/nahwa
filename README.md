# 나화 (裸花, nahwa)

AI 맞춤 꽃 추천 서비스 프로토타입

## About

나화는 꽃 구매가 어려운 고객을 위해 AI가 상황과 감정에 맞는 꽃을 추천해주는 서비스입니다.

- **AI 맞춤 추천**: 상황/감정 키워드 기반 꽃 조합 추천
- **프렌치 스타일**: 자연스럽고 꾸미지 않은 본질적 아름다움
- **예약제 운영**: 폐기율 최소화, 신선한 꽃 보장

## Tech Stack

- React 19 + Vite
- Tailwind CSS 4
- Vercel (Deploy)
